# Bài Tập Thực Hành ExpressJS và Github

Đây là project cơ bản dùng ExpressJS để hỗ trợ việc thực hành các thao tác với Github (commit, push, pull, nhánh, ...).

## Yêu cầu
- Node.js cài đặt sẵn trên máy

## Cách cài đặt
1. Clone repository về máy.
2. Mở terminal tại thư mục project và chạy lệnh cài đặt thư viện:
```bash
npm install
```

## Chạy project
Để chạy server, dùng lệnh sau:
```bash
npm run dev
```
(Ứng dụng sẽ tự động tải lại khi bạn thay đổi code).

Truy cập trên trình duyệt: [http://localhost:3000](http://localhost:3000)
